# {ConnectorName} デスティネーションSkill

オーケストレーター (`setup-pipeline.md`) から Read で読み込まれ、{ConnectorName} デスティネーション固有の処理を実行する。

## 必要環境変数

| 変数 | モードA | モードB | 説明 |
|------|---------|---------|------|
| `{PREFIX}_CONNECTION_ID` | 必須 | 不要 | 既存TROCCO接続ID |
| `{PREFIX}_...` | 不要 | 必須 | 認証情報（モードBで必要な変数を列挙） |

**モード判定:**
- `{PREFIX}_CONNECTION_ID` が設定済み → **モードA**（既存接続参照）
- `{PREFIX}_CONNECTION_ID` が空 → **モードB**（Terraformで新規作成）

## Step 3 (dest): 接続確認

### モードA: 既存接続ID参照

`{PREFIX}_CONNECTION_ID` が設定済み → そのIDを使用。API呼び出しスキップ。

### モードB: Terraform新規作成

`{PREFIX}_CONNECTION_ID` が空 → TROCCO API接続一覧を試行:

```bash
set -a && source .env.local && set +a
if [ -n "${PREFIX}_CONNECTION_ID" ]; then
  echo "{PREFIX}_CONNECTION_ID=${!PREFIX_CONNECTION_ID} (設定済み → このIDを使用)"
else
  RESULT=$(curl -s -w "\nHTTP_STATUS:%{http_code}" \
    "https://trocco.io/api/connections/{connector_name}" \
    -H "Authorization: Token ${TROCCO_API_KEY}")
  HTTP_CODE=$(echo "$RESULT" | tail -1 | sed 's/HTTP_STATUS://')
  BODY=$(echo "$RESULT" | sed '$d')

  if [ "$HTTP_CODE" = "200" ]; then
    echo "$BODY" | jq '.items[] | {id, name}' 2>/dev/null || echo "既存接続なし"
  elif [ "$HTTP_CODE" = "401" ] || [ "$HTTP_CODE" = "403" ]; then
    echo "API HTTP ${HTTP_CODE}: 接続一覧APIへのアクセス権限がありません"
    echo "→ TROCCO管理画面 > 接続情報 でIDを確認し .env.local に設定してください"
  else
    echo "API HTTP ${HTTP_CODE}: ${BODY}"
  fi
fi
```

## HCL情報

### output_option 構造

```
output_option_type = "{connector_name}"
output_option = {
  {connector_name}_output_option = {
    {connector_name}_connection_id = local.{connector_name}_connection_id
    # デスティネーション固有のオプション（database, table, mode等）
  }
}
```

### connection resource (モードB時)

```hcl
resource "trocco_connection" "{connector_name}_dest" {
  count = var.{prefix}_connection_id == null ? 1 : 0

  name            = var.{prefix}_connection_name
  connection_type = "{connector_name}"
  description     = "{ConnectorName} - auto-generated by TROCCO Pipeline Builder"

  # デスティネーション固有の認証フィールド
}

locals {
  {connector_name}_connection_id = (
    var.{prefix}_connection_id != null
    ? var.{prefix}_connection_id
    : trocco_connection.{connector_name}_dest[0].id
  )
}
```

### variables.tf ({ConnectorName}固有)

```hcl
variable "{prefix}_connection_id" {
  description = "既存TROCCO {ConnectorName}接続ID（設定時は接続作成をスキップ）"
  type        = number
  default     = null
}

variable "{prefix}_connection_name" {
  description = "TROCCO上の{ConnectorName}接続名（新規作成時のみ使用）"
  type        = string
  default     = "{connector_name}-auto"
}

# デスティネーション固有の変数を追加（database, schema, table, load_mode等）
```
